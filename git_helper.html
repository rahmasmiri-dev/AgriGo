📘 Guide Git pour Débutant

🎯 Objectif

Apprendre à créer et gérer un dépôt Git avec 5 collaborateurs, avec 1 branche par collaborateur.

🟢 1. Ouvrir un dossier local

cd chemin/vers/ton/dossier

🟡 2. Initialiser Git

git init

🔵 3. Lier au dépôt distant

git remote add origin https://github.com/user/NomProjet.git git remote -v

🟣 4. Ajouter et valider

git add . git commit -m "Initial commit"

🟠 5. Envoyer vers GitHub

git push -u origin main

🔴 6. Mettre à jour

git pull origin main git push

🟤 7. Créer une branche

git checkout -b feature-nom

Exemple :

git checkout -b feature-rahma
git checkout -b feature-gazzah
git checkout -b feature-sam
git checkout -b feature-wael
git checkout -b feature-amine

⚫ 8. Envoyer la branche

git push origin feature-nom

⚪ 9. Voir les branches

git branch git branch -r git branch -a

⚠️ Bonnes pratiques

  • Ne pas travailler sur main
  • Utiliser une branche personnelle
  • Faire plusieurs commits
  • Messages clairs

❌ Erreurs à éviter

  • Oublier git pull
  • Travailler sur la même branche
  • Ne pas commit régulièrement

🧠 Résumé

git init
git remote add origin URL
git add .
git commit -m "message"
git push
git pull
git checkout -b branche
git branch

🏁 Conclusion

Avec cette méthode :

  • ✔️ 1 dépôt partagé
  • ✔️ 5 branches
  • ✔️ Travail organisé sans conflit
